### ASP.NET Core で静的ファイルへのアクセス時に HTTP ヘッダーに "Cache-Control" を付加して一定期間リクエストを軽減させる方法

Startup.cs の Configure() 内で UseStaticFiles() のオプションで **レスポンスのヘッダーに "Cache-Control" を追加** します。
